TikTok Integration Demo

Creator-authorized upload workflow for Fox's After-School Lab.

Sandbox review flow · SELF_ONLY test upload
1. Login Kit

The creator connects their TikTok account through OAuth using user.info.basic and video.publish.

2. Creator Info

The app checks posting options such as available privacy levels. Test publishing uses SELF_ONLY.

3. Creator Review

The creator reviews the final MP4, caption, hashtags, and privacy setting before upload.

4. Content Posting API

The app initializes Direct Post, uploads the selected MP4, and stores the returned publish ID locally.
